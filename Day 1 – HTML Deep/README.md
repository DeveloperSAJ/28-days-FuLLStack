#  HTML Mastery Repository

## Overview

This repository contains everything I have learned and practiced in **HTML (HyperText Markup Language)**. It reflects my complete journey from basic structure to advanced semantic layouts and real-world webpage building.

The goal of this repository is to build a strong foundation in web development by mastering HTML concepts and implementing them through practice and exercises.

## What This Repository Includes

### Core Fundamentals

* Basic HTML document structure
* Headings, paragraphs, and formatting tags
* Lists (ordered, unordered, description lists)
* Links and navigation
* Images and media embedding

### Forms & Inputs

* Input types (text, email, password, number, etc.)
* Radio buttons & checkboxes
* Dropdown menus
* Textareas
* Form validation basics
* Labels and accessibility practices

### Semantic HTML

* `<header>`
* `<nav>`
* `<section>`
* `<article>`
* `<aside>`
* `<footer>`

### Tables

* Table structure
* Thead, tbody, tfoot
* Rowspan & colspan
